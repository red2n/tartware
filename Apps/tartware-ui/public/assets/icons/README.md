# PWA Icons Required

Generate icons in the following sizes and place them in this directory:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You can use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- ImageMagick: `convert logo.png -resize 512x512 icon-512x512.png`

All icons should be maskable and work with any background color.
