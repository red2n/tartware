# Guest Profile Management

## Profile Data Elements

### Identity & Contact

| Field | Type | Purpose | Sensitivity |
|-------|------|---------|-------------|
| Full name | Required | Identification | PII |
| Date of birth | Optional | Age verification, marketing | PII |
| Gender | Optional | Salutation, preferences | PII |
| Nationality | Optional | Language, customs | PII |
| Email | Required | Communication | PII |
| Phone | Required | Contact, SMS | PII |
| Address | Optional | Billing, marketing | PII |

### Documents

| Document | When Required | Retention |
|----------|---------------|-----------|
| Passport | International guests | Per local law |
| Driver's license | Domestic guests | Per local law |
| Government ID | All check-ins | Per local law |
| Credit card | Payment guarantee | Tokenized only |
| Company ID | Corporate travelers | Duration of stay |

### Preferences

| Category | Examples |
|----------|----------|
| Room | High floor, quiet, away from elevator |
| Bedding | King preferred, extra pillows |
| Amenities | Feather-free, hypoallergenic |
| F&B | Dietary restrictions, allergies |
| Communication | Email preferred, no calls |
| Newspaper | Wall Street Journal, local |
| Temperature | Prefer cool (68°F / 20°C) |

## Loyalty Program Tiers

| Tier | Requirements | Benefits |
|------|--------------|----------|
| **Base/Member** | Sign up | Member rate, points earning |
| **Silver** | 10-20 nights/year | Early check-in, late checkout |
| **Gold** | 25-50 nights/year | Room upgrade, bonus points |
| **Platinum** | 50-75 nights/year | Suite upgrade, lounge access |
| **Elite/Ambassador** | 75+ nights/year | Guaranteed availability, host |

### Points Economics

| Activity | Typical Earning | Redemption |
|----------|-----------------|------------|
| Room night | 10 points/$1 | 25,000 points = 1 night |
| F&B spend | 5 points/$1 | Varies by tier |
| Partner (airline) | 500 miles/stay | Transfer ratios vary |
| Credit card | 2-5x points | Co-brand multipliers |

## Guest Communication

### Automated Triggers

| Event | Communication | Timing |
|-------|---------------|--------|
| Booking confirmed | Confirmation email | Immediate |
| Pre-arrival | Upsell, info request | 3-7 days before |
| Day of arrival | Welcome, directions | Morning of |
| During stay | Service offers | Based on stay |
| Checkout | Folio, thank you | At departure |
| Post-stay | Review request | 1-3 days after |
| Birthday | Special offer | Annual |
| Anniversary | Return offer | Annual |

### Communication Preferences

| Preference | Options |
|------------|---------|
| Channel | Email, SMS, app push, none |
| Frequency | Every booking, monthly, quarterly |
| Content | Offers, info only, none |
| Language | Based on profile or property default |

## Guest History

### Stay History

| Data Point | Use Case |
|------------|----------|
| Total stays | Loyalty recognition |
| Total spend | Value classification |
| Average rate | Pricing strategy |
| Room type pattern | Preference prediction |
| Length of stay pattern | Package offers |
| Booking source | Channel attribution |

### Guest Score/Value

| Segment | Criteria | Treatment |
|---------|----------|-----------|
| High value | Top 10% spend | Premium service, manager greeting |
| Medium value | 60th-90th percentile | Standard plus recognition |
| Low value | Bottom 60% | Standard service |
| High potential | New + high rate | Cultivate relationship |
| At risk | Declining frequency | Win-back campaign |

## Privacy & Consent

### GDPR Rights

| Right | Implementation |
|-------|----------------|
| Access | Guest can view all data |
| Rectification | Guest can correct data |
| Erasure | Right to be forgotten |
| Portability | Export in machine-readable |
| Object | Opt-out of marketing |

### Consent Management

| Purpose | Consent Required | Basis |
|---------|------------------|-------|
| Reservation processing | No | Contract |
| Stay delivery | No | Contract |
| Marketing email | Yes | Consent |
| Third-party sharing | Yes | Consent |
| Analytics | Depends | Legitimate interest |
