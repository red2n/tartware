# Loyalty Programs

## Program Structure

### Tier Systems

| Tier Structure | Description | Example Brands |
|----------------|-------------|----------------|
| **4-tier** | Base, Silver, Gold, Platinum | Marriott Bonvoy |
| **5-tier** | Base through Ambassador | Hilton Honors |
| **3-tier** | Member, Elite, Premier | IHG Rewards |
| **Status match** | Cross-program recognition | Industry practice |

### Qualification Methods

| Method | Description |
|--------|-------------|
| **Nights** | Number of qualifying nights |
| **Stays** | Number of separate reservations |
| **Points** | Points earned from spend |
| **Spend** | Total eligible revenue |
| **Elite nights** | Rollover for lifetime status |

## Points & Currency

### Earning Structure

| Transaction | Typical Earn Rate |
|-------------|-------------------|
| Hotel room revenue | 10 points per $1 |
| Restaurant spend | 5 points per $1 |
| Spa services | 5 points per $1 |
| Co-brand credit card | 2-6x points |
| Partner airline | 500-2000 miles/stay |
| Car rental partner | 50-500 points |

### Redemption Options

| Option | Value | Availability |
|--------|-------|--------------|
| Free night award | 25,000-100,000 pts | Category-based |
| Points + cash | Variable | Limited |
| Room upgrade | 5,000-15,000 pts | Subject to availability |
| Experience rewards | Variable | Partnership dependent |
| Airline miles | Transfer | Usually 1:1 to 3:1 |
| Merchandise | Fixed catalog | Lower value |

### Points Valuation

| Brand | Approx. Value (USD) |
|-------|---------------------|
| Marriott Bonvoy | 0.7 - 1.0¢/point |
| Hilton Honors | 0.4 - 0.6¢/point |
| IHG Rewards | 0.5 - 0.7¢/point |
| World of Hyatt | 1.5 - 2.0¢/point |

## Program Economics

### Cost Considerations

| Element | Typical Cost |
|---------|--------------|
| Point redemption liability | 0.5-1.0¢ per point |
| Upgrade provision | $15-50 per delivered |
| Lounge access | $35-75 per visit |
| Welcome amenity | $5-20 per stay |
| Breakfast comp | $15-45 per person |

### Revenue Impact

| Metric | Impact |
|--------|--------|
| Member booking premium | +10-20% ADR willingness |
| Return frequency | +30% repeat stays |
| Ancillary spend | +25% F&B/spa spend |
| Direct booking | +40% direct vs. OTA |
