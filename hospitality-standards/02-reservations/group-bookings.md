# Group Bookings

## Group Definition

A **group booking** is typically defined as:
- **10 or more rooms** on the same night(s)
- Common purpose (meeting, wedding, tour)
- Shared billing arrangement
- Negotiated rates below rack

## Group Types

| Type | Code | Characteristics | Billing |
|------|------|-----------------|---------|
| **Corporate Meeting** | CORP | Weekday, meeting rooms, AV | Direct bill |

## Room Block Management

### Block Status

| Status | Description |
|--------|-------------|
| **Tentative** | Held pending contract |
| **Definite** | Contract signed, committed |
| **Picked Up** | Rooms assigned to guests |
| **Released** | Returned to general inventory |
| **Cutoff** | Deadline passed, unused released |

### Attrition

Attrition clause protects hotel from under-pickup:

| Clause | Typical Terms |
|--------|---------------|
| Allowable attrition | 10-20% of block |
| Attrition fee | 80-90% of room rate × unused rooms |
| Cutoff date | 14-30 days before arrival |

### Room Block Example

```
Original Block:     50 rooms/night × 3 nights = 150 room nights
Allowable Attrition: 15% = 22 room nights
Minimum Commitment:  128 room nights

At Cutoff:
  Picked up:        45 rooms/night = 135 room nights ✓ (above minimum)

OR

At Cutoff:
  Picked up:        35 rooms/night = 105 room nights
  Shortfall:        128 - 105 = 23 room nights
  Attrition Fee:    23 × $150 × 85% = $2,932.50
```

## Group Pricing

| Element | Strategy |
|---------|----------|
| Room rate | 10-30% below BAR |
| Complimentary rooms | 1 comp per 40-50 paid (varies) |
| Meeting space | Waived with room commitment |
| F&B minimum | Required for event space |
| Parking | Often included or discounted |

## Rooming List

Required information for each group guest:

| Field | Required | Notes |
|-------|----------|-------|
| Guest name | Yes | As on ID |
| Arrival date | Yes | May vary within block |
| Departure date | Yes | May vary within block |
| Room type | Yes | Per contracted allocation |
| Sharing with | If applicable | Room sharing pairs |
| Special requests | No | ADA, preferences |
| Payment method | Yes | Individual or master |

### Rooming List Deadlines

| Group Size | Typical Deadline |
|------------|------------------|
| 10-25 rooms | 7 days prior |
| 26-100 rooms | 14 days prior |
| 101-300 rooms | 21 days prior |
| 300+ rooms | 30 days prior |
